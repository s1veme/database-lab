# App Database

# Установка
- Скопируйте проект
    ```console
    $ git clone ...
    ```
- Скопируйте `.env`, либо создайте руками
    ```console
    $ cp .env.example .env
    ```
- Установите параметры в файле `.env`
- Запустите контейнер
    ```console
    $ docker-compose up -d --build
    ```
